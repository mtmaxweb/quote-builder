A React-based quote builder for WordPress.

== Description ==

This plugin embeds a React-based quote builder application into your WordPress site. Use the shortcode [react_quote_builder] to display the quote builder on any page or post.

== Installation ==

1. Upload the `react-quote-builder` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Place the shortcode [react_quote_builder] in your pages or posts where you want the quote builder to appear.
